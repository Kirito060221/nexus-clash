# NEXUS CLASH

LoL（League of Legends）の仕組みを再現したブラウザ用の5v5ミニMOBA。
ライブラリ・ビルドツール不要、**`index.html` 1ファイルだけで動く**（HTML5 Canvas + 素のJavaScript）。

自分1人 + 味方AI 4体 vs 敵AI 5体で対戦し、敵ネクサスの破壊を目指す。

## 遊び方

`index.html` をブラウザで開くだけ（PC・マウス+キーボード推奨）。
GitHub Pagesで公開すればURLを共有するだけで友人も遊べる（下記「公開手順」）。

### 操作

| 入力 | 動作 |
|------|------|
| クリック（右/左どちらでも） | 移動 / 敵をクリックで攻撃 |
| `Q` `W` `R` | スキル（マウス方向に発動） |
| `B` | リコール（拠点に帰還・被弾で中断） |
| `P` | ショップ（自軍拠点内でのみ開ける） |
| `Tab` | スコアボード |

### 目的

ミニオンをラストヒットしてゴールドと経験値を稼ぎ、アイテムで強化しながら
3レーンのタワーを折り、最奥のネクサスを破壊すれば勝利。

## 再現しているLoL要素

- 3レーン + 川 + ブッシュ（茂みの中は敵から見えない）
- 一定間隔で湧くミニオンウェーブ、ラストヒットによるCS / ゴールド獲得
- 経験値・レベル（最大18）・スキルのクールダウンとマナ消費
- タワー2段階（外側が健在の間は内側が無敵。味方を攻撃した敵を狙うタワーアグロ）
- ネクサス破壊で決着、拠点ショップとアイテム6枠
- リコール、噴水回復と敵噴水のレーザー
- キルフィード / ファーストブラッド / アシスト / ミニマップ / スコアボード
- レーンを押し引きし、低HPで撤退→買い物→復帰まで行う自律AI

## チャンピオン（オリジナル4体）

| 名前 | ロール | 特徴 |
|------|--------|------|
| アオイ（葵） | メイジ | 遠距離の魔法弾＋ワープ＋直線レーザー |
| レンジ（連） | マークスマン | 貫通矢＋攻撃速度バフ＋超長距離狙撃 |
| ガンテツ（岩） | タンク | 範囲ダメージ＋シールド＋跳躍スタン |
| シノブ（忍） | アサシン | 突進＋ステルス＋低HP処刑 |

## 公開手順（GitHub Pages）

このフォルダをそのままリポジトリにして push すれば、静的サイトとして配れる。

```bash
# このフォルダの中で
git init
git add .
git commit -m "Initial commit: NEXUS CLASH"
git branch -M main
git remote add origin https://github.com/<ユーザー名>/nexus-clash.git
git push -u origin main
```

その後、GitHub のリポジトリページで:

1. **Settings** → 左メニュー **Pages**
2. **Source** を「Deploy from a branch」にする
3. Branch を **main / (root)** にして **Save**
4. 数十秒〜数分後、`https://<ユーザー名>.github.io/nexus-clash/` で公開される

このURLを友人に送れば、インストール不要でそのまま遊んでもらえる。

## 技術メモ

- 依存ゼロ。単一HTML内にCSS/JSを同梱
- ゲームロジックは `window.NX` にテスト用フックを公開（`NX.startGame('aoi')` 等）しており、ヘッドレスでシミュレーション可能
- 座標系はワールド3200×3200。カメラがプレイヤーを追従
